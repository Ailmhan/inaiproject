# Генерация иконок для PWA

Для полноценной работы PWA и iOS Web View необходимо создать следующие иконки:

## Требуемые иконки:

1. **icon-192x192.png** - 192x192 пикселей (для Android и PWA)
2. **icon-512x512.png** - 512x512 пикселей (для Android и PWA)
3. **apple-touch-icon.png** - 180x180 пикселей (для iOS)

## Способы создания иконок:

### Вариант 1: Онлайн генераторы
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Builder](https://www.pwabuilder.com/imageGenerator)

### Вариант 2: Использование существующего логотипа
Если у вас есть логотип в `public/assets/icons/composio-logo-*.svg`, вы можете:
1. Открыть SVG в графическом редакторе (Figma, Adobe Illustrator, Inkscape)
2. Экспортировать в PNG с нужными размерами
3. Сохранить в папку `public/`

### Вариант 3: Создание простой иконки
Можно создать простую иконку с текстом "CC" (Composio Chat) или использовать первый символ названия.

## Рекомендации:
- Используйте квадратные иконки (без скругления - iOS добавит его автоматически)
- Фон должен быть непрозрачным
- Избегайте мелких деталей - иконки будут отображаться в маленьком размере
- Используйте контрастные цвета для лучшей видимости

## После создания иконок:
1. Поместите файлы в папку `public/`
2. Убедитесь, что файлы названы точно так, как указано выше
3. Перезапустите dev-сервер для применения изменений
